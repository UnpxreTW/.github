# 取得協助 · Getting Help

## 繁體中文

請依問題類型選擇合適的管道：

1. **Bug 回報、功能建議** → 對應儲存庫的 Issues 頁面
2. **使用問題、討論** → 該儲存庫若有開啟 Discussions，請在 Discussions 發問
3. **私下事項、商業聯繫** → **mail@unpxre.me**

回應時效為 best-effort，不保證 SLA。安全性弱點請見 [SECURITY.md](./SECURITY.md)。

## English

Please choose the appropriate channel based on your issue:

1. **Bug reports, feature requests** → the repository's Issues page
2. **Usage questions, discussions** → the repository's Discussions tab (if enabled)
3. **Private matters, business inquiries** → **mail@unpxre.me**

Response time is best-effort; no SLA is guaranteed. For security vulnerabilities, see [SECURITY.md](./SECURITY.md).
